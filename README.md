# 🛍️ Lavechia Store - E-commerce de Loja de Roupas

## 📖 Sobre o projeto

O **Lavechia Store** é um sistema de e-commerce desenvolvido como projeto acadêmico do curso de **Gestão da Tecnologia da Informação (GTI)** da FATEC Franco da Rocha.

O objetivo do projeto foi desenvolver uma plataforma web para automatizar os processos de uma loja de roupas, centralizando o gerenciamento de produtos, estoque, vendas e clientes em um único ambiente, além de melhorar a experiência de compra dos usuários.

A aplicação foi criada para solucionar problemas encontrados no modelo de vendas realizado exclusivamente por redes sociais, como excesso de tarefas manuais, inconsistência nos registros e dificuldades na gestão do negócio.

---

## 🎯 Objetivos do projeto

- Automatizar tarefas repetitivas do gestor da loja;
- Melhorar a experiência de compra dos clientes;
- Centralizar o gerenciamento do negócio;
- Disponibilizar relatórios administrativos;
- Integrar sistemas de pagamento e cálculo de frete;
- Desenvolver uma interface responsiva e intuitiva.

## 💻 Como executar o projeto

### Pré-requisitos

- PHP 7 ou superior
- MySQL
- Apache (XAMPP, WAMP ou Laragon)

### Instalação

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/lavechia-store.git
```

2. Coloque o projeto na pasta do servidor web.

3. Crie um banco de dados MySQL.

4. Importe o arquivo `.sql` do projeto.

5. Configure a conexão com o banco de dados.

6. Inicie o Apache e o MySQL.

7. Acesse:

```text
http://localhost/lavechia-store
```
